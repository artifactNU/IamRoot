# Logrotate Configuration Examples

Example logrotate configurations for log management.

---

## Purpose

Provide templates for:
- Application log rotation
- Custom log file handling
- Compression and retention policies
- Post-rotation scripts

---

## Guidelines

All examples must:
- Use `.example` suffix
- Include comments explaining directives
- Consider disk space constraints
- Be safe for production adaptation
- Document any external dependencies

---

## Example Topics

Future examples may include:
- Basic rotation with compression
- Multiple log file patterns
- Rotation with post-rotate hooks
- Size-based vs time-based rotation
- Integration with systemd-journald
