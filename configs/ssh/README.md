# SSH Configuration Examples

Example SSH configurations for secure remote access.

---

## Purpose

Provide safe, well-documented SSH configuration templates for:
- Client configuration (`~/.ssh/config`)
- Server configuration (`/etc/ssh/sshd_config`)
- Key management patterns

---

## Guidelines

All examples must:
- Use `.example` suffix
- Include comments explaining options
- Prioritize security over convenience
- Document compatibility considerations
- Never include private keys or credentials

---

## Example Topics

Future examples may include:
- Hardened sshd_config
- Client config with jump hosts
- Key-based authentication setup
- Port forwarding patterns
