# Security Scripts

Scripts for security auditing, hardening assessment, forensics, and log analysis.

---

## Contents

- Login auditing (failed attempts, successful logins)
- Sudo usage analysis
- System hardening assessment
- Incident evidence collection
- Syslog and cron security review

---

## Critical Considerations

- Output may contain sensitive system information — treat as confidential
- Consider legal and policy implications before collecting forensic data
- Document data retention requirements for collected evidence
- Tools here assess and report; they do not modify system security controls
